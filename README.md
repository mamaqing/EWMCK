## GitHub Pages
要启用 GitHub Pages：
1. 在仓库页面点击 Settings -> Pages
2. 在 Branch 下拉菜单选择包含 README.md 的分支（如 `main` 或 `gh-pages`），通常选择 `main` 并点击 Save
3. 等待几分钟，页面会显示站点地址（https://mamaqing.github.io/EWMCK/）

> 提示：如果 Branch 下拉菜单初次看不到分支，请先在 Code 页通过 “Add file” 添加 README.md 到目标分支，然后返回 Settings 再选择分支。
